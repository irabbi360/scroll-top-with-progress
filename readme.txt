Scroll Top With Progress is a fully customizable plugin that adds a Back to Top button with a scroll progress indicator to your WordPress website. Enhance user experience and make navigation smoother with this lightweight plugin.

== Description ==

**Scroll Top With Progress** allows you to easily add a floating Back to Top button on your website. It includes a scroll progress indicator around the button and offers complete customization options:

* Change primary and secondary colors of the progress circle.
* Customize the icon color.
* Choose the position of the button (left or right corner).
* Reset colors to default.
* Live preview customization in WordPress Customizer.

**Features:**

* Fully responsive Back to Top button.
* Conic gradient progress indicator.
* Customizable colors and position.
* Lightweight and fast.
* Admin panel to configure options.
* Works with any theme.
* Translation-ready.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/scroll-top-with-progress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Scroll To Top** in your WordPress admin menu to configure colors and button position.
4. Optionally, customize the button appearance via the WordPress Customizer.

== Frequently Asked Questions ==

= Can I change the button position? =

Yes, you can choose either **left** or **right** position from the plugin settings page or via the Customizer.

= Can I reset the colors to default? =

Yes, there is a **Reset to Default Colors** button in the plugin settings.

= Will it work with any theme? =

Yes, this plugin is fully compatible with all modern WordPress themes.
